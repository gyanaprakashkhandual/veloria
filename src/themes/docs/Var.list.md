# 💡 Naming Notes

* ❌ `fifer` → ✅ `quinary`
* ❌ `quertiaty` → ✅ `quaternary`
* Keep pattern: **primary → secondary → tertiary → quaternary → quinary**
* Use **semantic names**, not colors (no `--red-500` in app-level theme)
